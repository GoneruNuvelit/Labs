## Задание A1: Удаление текста в скобках

Напишите программу, которая укорачивает входной текст, удалив из текста все, что написано в круглых скобках, включая сами скобки. Для решения задачи реализуйте функцию, которая получает на вход исходный текст, а возвращает укороченный.

**Пример работы программы:**

**Исходный текст:** Падал (куда он там падал) прошлогодний (значит очень старый) снег (а почему не дождь) (! (!)).
**Укороченный текст:** Падал прошлогодний снег.

**Рекомендации:**
- Для поиска в исходном тексте скобок используйте функции `find()` или `rfind()`
- Для замены фрагмента текста на пустую строку используйте функцию `replace()`

## Задание A2: Разделение текста на предложения

Дан текст, состоящий из предложений. Предложения текста заканчиваются символами точка, восклицательный или вопросительный знак. Найдите количество предложений в тексте и выведите все предложения на экран, по одному в каждой строке.

**Пример работы программы:**

**Ввод:** Не jests at scars. That never felt a wound! Hello, friend! Are you OK?
**Вывод:**
He jests at scars.
That never felt a wound!
Hello, friend!
Are you OK?
Предложений в тексте: 4

**Рекомендации:**
- Учтите, что между предложениями может быть более одного пробельного символа
- Лишние пробелы выводить не надо
- Для разделения текста можно использовать регулярные выражения

## Задание A3: Создание аббревиатуры

Исходный текст состоит из слов, разделенных пробелом. Требуется получить строку, которая состоит только из первых букв каждого слова (аббревиатура).

**Дополнительные условия:**
- Все буквы аббревиатуры должны быть в верхнем регистре
- В аббревиатуру не включаются первые буквы слов, длина которых меньше 3

**Пример работы программы:**

**Ввод:** New York City
**Вывод:** NYC

**Ввод:** Yanka Kupala State University of Grodno
**Вывод:** YKSUG

## Задание A4: Генетический поиск

Белок представляет собой большую молекулу, вырабатываемую в клетке живого организма. Каждый белок состоит из аминокислот, соединенных в цепочку. Для простоты понимания генетики каждой из 20 аминокислот присвоена буква алфавита.

**Входные данные:**
- Файл `sequences.txt` содержит генетические данные
- Файл `commands.txt` содержит команды для выполнения над этими данными

**Операции:**
1. **search** - найти указанную последовательность аминокислот
2. **diff** - определить количество аминокислот, которыми различаются два белка
3. **mode** - найти наиболее часто встречающуюся аминокислоту в указанном белке

**Требования:**
- Реализовать выполнение всех операций из файла `commands.txt`
- Результаты записать в файл `genedata.txt`
- Учесть специальные случаи (белки не найдены и т.д.)
- Соблюсти формат выходного файла

**Пример формата выходного файла:**
```
Ivan Ivanov
Генетический поиск
---
001 mode PREDICTED: similar to double homeobox
amino-acid occurs:
R    17
---
002 search LKSQ
organism    protein
Homo sapiens (Human)   TRAF interacting protein TANK isoform b
```
